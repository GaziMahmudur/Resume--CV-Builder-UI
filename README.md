# 🚀 ResumeForge | Professional CV Builder

**ResumeForge** is a high-performance, single-file web application designed to help users create stunning, professional resumes in minutes. Built with pure vanilla technologies, it offers a seamless, real-time editing experience with a focus on typography, layout precision, and ease of use.

## ✨ Features

### 🎨 Premium Design Templates
Choose from three professionally crafted templates designed for different career stages:
- **Classic Dark:** Bold navy header, professional single-column layout.
- **Editorial Warm:** Elegant two-column layout with a warm sidebar and coral accents.
- **Clean White:** Minimalist, high-whitespace design optimized for ATS (Applicant Tracking Systems).

### ⚡ Real-Time Preview
Watch your CV come to life as you type. The application features a debounced rendering engine that updates the A4 preview panel instantly without flicker.

### 📄 Professional PDF Export
Export your resume as a pixel-perfect A4 PDF using the integrated `html2pdf.js` library. The export handles scaling, high-DPI rendering, and cross-origin images flawlessly.

### 💾 Auto-Save & Persistence
Never lose your progress. All data is automatically serialized and saved to your browser's `localStorage`. Use the "Clear Data" option to start fresh.

### 🛠️ Advanced Form Control
- **Collapsible Entries:** Manage multiple work experiences and education blocks with ease.
- **Tag System:** Add skills, technologies, and interests using an interactive pill-tag input.
- **Photo Support:** Upload and preview your profile picture instantly.
- **Character Counters:** Ensure your professional summary stays concise.

---

## 🛠️ Tech Stack

- **HTML5:** Semantic structure.
- **CSS3:** Custom variables, Flexbox/Grid layout, and smooth transitions.
- **JavaScript (ES6+):** Vanilla logic for state management and DOM manipulation.
- **html2pdf.js:** Client-side PDF generation.
- **Google Fonts:** "Inter" for the application UI and "Lora" for elegant resume typography.
